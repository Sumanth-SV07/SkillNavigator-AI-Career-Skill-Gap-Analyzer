# SkillNavigator – AI Career Skill Gap Analyzer

SkillNavigator is an AI-powered web application that helps students analyze their current skills and identify the skills required for their desired career roles.

The system compares user skills with industry-required skills and provides:

• Skill gap analysis  
• Skill priority recommendations  
• Learning resources  
• Career roadmap  
• Progress score visualization  

---

## Features

- Resume / skill analysis
- Career role selection
- Skill gap detection
- Skill priority recommendation
- Learning resources for missing skills
- Career roadmap guidance
- Data visualization using graphs

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Matplotlib

---

## How to Run the Project

1. Install Python
2. Install required libraries "pip install flask matplotlib"
3. Run the application "python app.py"
4. Open in browser "http://127.0.0.1:5000"

---

## Project Goal

The goal of this project is to help students understand their skill gaps and guide them toward the right career path using intelligent recommendations.
